## 🛠️ Setup and Usage

1. **Clone the repository:**
    ```bash
    git clone https://github.com/Syed-Abid/Web-Scraper-in-Python.git
    ```

2. **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3. **Run the Web Scraper:**
    ```bash
    python web_scraper.py
    ```
